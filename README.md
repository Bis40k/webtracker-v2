# WebTracker 🚀

GPS-трекер на веб-приложении. Отслеживайте свои устройства через браузер, используя Макродроид для отправки координат с Android.

## 🎯 Главное

- 📍 Карта в реальном времени
- 📧 Вход по email (Magic Link)
- 📱 Обновления с Макродроида
- 🔐 Защита данных (только ваши маячки видны)

## 🚀 Быстрый старт

```bash
npm install
npm run dev
```

Откройте http://localhost:3000
